# ZK API Credits — Smart Contracts

Solidity contracts for the ZK API Credits system, built with Hardhat + SE2.

## Contracts

| Contract | Description |
|----------|-------------|
| `APICredits.sol` | Core contract — staking, commitment registration, Merkle tree, server claims |
| `Verifier.sol` | Auto-generated UltraHonk verifier from the Noir circuit |

## Deploy (Local)

```bash
yarn chain        # start local Hardhat node
yarn deploy       # deploy all contracts
```

## Deploy (Base Mainnet)

```bash
# Set deployer key in .env
yarn deploy --network base
```

## Deploy (Sepolia Testnet)

```bash
yarn deploy --network sepolia
```

## Verify on Etherscan

```bash
yarn verify --network base
```

## Key Details

- **PRICE_PER_CREDIT**: 0.001 ETH per API credit
- **Merkle Tree**: LeanIMT (Poseidon2 hashing, up to 65,536 leaves)
- **Verifier**: UltraHonk (generated by Barretenberg from the Noir circuit)
